# Flex Component

Layout component with props that manipulate flex properties.

```js
  <Flex
    grow={1} // flex-grow
    shrink={0} // flex-shrink
    basis={0} // flex-basis
    justify='center' // justify-content
    items='center' // align-items
    content='stretch' // align-content
    self='stretch' // align-self
    wrap // wrap
  />

```
